# Frontend Expert AI Agent

A simple Python AI agent using the **AI Agent SDK** that acts as a **frontend development expert**.  
It can respond to queries, provide guidance, and assist with frontend tasks using the Gemini AI model.

---

## Features

- Interacts in natural language as a frontend development expert.
- Provides suggestions and guidance for HTML, CSS, and JavaScript.
- Uses **Gemini 2.0 Flash** model for AI completions.
- Built with **AI Agent SDK** for easy customization and integration.

---

Hello! I am your frontend Expert. How can I assist you today?
You can change the prompt by modifying the input in Runner.run_sync:

result = Runner.run_sync(agent, input="Hello what are you doing", run_config=config)
